# snowbros

Deterministic engineering intelligence for JavaScript/TypeScript codebases.

This is a convenience package that forwards to
[`@snowbros/atlas`](https://www.npmjs.com/package/@snowbros/atlas) — the
primary package. Both expose the `sb` and `snowbros` commands.

```sh
npx snowbros analyze
```

Supported platforms: Windows x64, macOS x64/arm64, Linux x64/arm64.
On other platforms, build from source: `cargo install snowbros-atlas --locked`.

Full documentation:
[github.com/snowbros-labs/atlas](https://github.com/snowbros-labs/atlas)
