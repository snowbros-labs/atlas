# Extension assets

- `icon.png` — 128×128 Marketplace icon (wired in `package.json` as `"icon"`),
  rendered from the Snowbros Atlas logo.
- `icon.svg` / `icon-dark.svg` — the vector logo (light/dark), source of truth.

## Optional before publish

A gallery banner image is optional (`galleryBanner.color` is already set). Add a
`banner.png` here and reference it if desired. Marketplace screenshots can be
taken from the extension in a real editor — see `../DEVELOPMENT.md`.
