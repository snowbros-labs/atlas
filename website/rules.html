Severity and confidence

Severity says how much a finding matters; confidence says how sure the engine is. Certain findings are structural facts (a cycle exists or it doesn't). Possible findings are honest hedges — a "dead" file might be loaded by a runtime convention the graph can't see, so it reports Low/Possible, and one config line silences the category if it's noise for your project.

Secrets are redacted

Hardcoded-secret findings never print the value — a four-character redacted preview only, in every output format including SARIF and JSON.
