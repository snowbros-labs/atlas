Installation

Every method installs two binaries: snowbros and the alias sb. Prebuilt: Windows x64, macOS x64/arm64, Linux x64/arm64. All archives ship SHA-256 checksums.

npm / npx (Node ≥ 18)

npx snowbros analyze        # one-shot
npm install -g @snowbros/atlas  # global
npm install -D @snowbros/atlas  # per-project

Shell installer — macOS, Linux

curl --proto '=https' --tlsv1.2 -LsSf \
  https://github.com/snowbros-labs/atlas/releases/latest/download/snowbros-atlas-installer.sh | sh

PowerShell installer — Windows

irm https://github.com/snowbros-labs/atlas/releases/latest/download/snowbros-atlas-installer.ps1 | iex

Homebrew — macOS, Linux

brew install snowbros-labs/tap/snowbros-atlas

Cargo — from source

cargo install snowbros-atlas --locked

Verify

sb --version
cd your-project && sb init && sb analyze

Editor (LSP)

Command sb, argument lsp, stdio transport. Neovim and Helix snippets in the install guide.
