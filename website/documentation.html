Commands

CommandWhat it does
sb initWrite a starter snowbros.toml
sb analyzeFull analysis; --format terminal|json|markdown|sarif|html
sb analyze --ciExit 2 when High+ findings exist
sb analyze --no-cacheForce a cold run
sb watchContinuous analysis; prints only deltas
sb fixDeterministic auto-fixes; --dry-run, --rule, --file
sb graph --format dotExport semantic graph (Graphviz)
sb explain RULE_IDRule logic + false-positive guards
sb lspLSP server over stdio

CI recipe (GitHub Actions)

- run: npx snowbros analyze --format sarif > snowbros.sarif
- uses: github/codeql-action/upload-sarif@v3
  with:
    sarif_file: snowbros.sarif
- run: npx snowbros analyze --ci   # gate on High+ findings

Configuration

[analysis]
min_severity = "low"          # info|low|medium|high|critical
min_confidence = "possible"   # possible|likely|certain

[rules]
disable = ["exports/*", "graph/dead-file"]
enable  = ["exports/unused-export"]   # enable > disable > thresholds
