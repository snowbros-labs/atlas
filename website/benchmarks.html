Why warm runs are trustworthy

The incremental cache (xxh3 content hashes + mtime fingerprints, invalidated by any config or version change) can only skip parsing — never change results. An end-to-end test asserts warm output is byte-identical to a cold run. If you ever find a divergence, that's a release-blocking bug.

Reproduce

git clone --depth 1 https://github.com/axios/axios && cd axios
sb analyze            # cold
sb analyze            # warm
cargo bench -p snowbros   # criterion suite (from the SNOWBROS repo)
